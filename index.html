Pseudo-Elements Make the Web Sexier

Hint: You’re Already Using Them

A Simple Example: The Clearfix


<div class="wrapper">
    <div class="left">
        <p>I'm on the left!</p>
        <p>Let's get weird!</p>
    </div>
    <div class="right">
        <p>I'm on the right?</p>
    </div>
</div>

This Looks Wrong


.wrapper {
    display: block;
    width: 100%;
    background: red;
}
.left,.right { background: #bcf; }
.left        { float: left; }
.right       { float: right; }

I'm on the left!

Let's get weird!

I'm on the right?

I'll have no part in this.

Much Better


.wrapper {
    display: block;
    width: 100%;
    background: red;
}
.wrapper:after {
    content: ' ';
    display: block;
    clear: both;
}
.left,.right { background: #bcf; }
.left        { float: left; }
.right       { float: right; }

I’m on the left!

Let’s get weird!

I’m on the right?

I’ll have no part in this.

Look familiar? :after is a pseudo-element.

Drop Caps with :first-letter


p:first-of-type:first-letter {
  float: left;
  margin: 57px 8px 8px 0;
  font-family: 'Cheshire Initials';
  font-size: 4em;
}

Detect Screen Size


body:after {
    content: 'desktop';
    visibility: hidden;
    position: absolute;
    top: 0; left: 0;
    width: 0; height: 0;
    z-index: -1;
    pointer-events: none;

    @media (max-width: 1024px) {
        content: 'tablet';
    }

    @media (max-width: 600px) {
        content: 'mobile';
    }
}
